# Cointracker

## To install

To clone the repo:
`git clone https://github.com/Reblixt/Cointracker.git`

Navigate to Project:
`cd Cointracker`

Install dependencies:
`npm i`

Run it:
`npm run dev`

The API comes from:
`https://mobula.io/`
